Recurrencia y Árboles

Fundamentos y aplicaciones de algoritmos recursivos

¿Qué es la Recurrencia?

La recurrencia es una técnica de programación donde una función se llama a sí misma para resolver un problema más pequeño del mismo tipo. Es fundamental en el diseño de algoritmos y especialmente útil para trabajar con estructuras de datos como árboles.

Componentes de una función recursiva:

  • Caso base: Condición que detiene la recursión
  • Caso recursivo: La función se llama a sí misma con un problema más pequeño

Ejemplo: Factorial

def factorial(n):
    # Caso base
    if n <= 1:
        return 1
    # Caso recursivo
    return n * factorial(n - 1)

# Ejemplo de uso
print("Factorial de 5:", factorial(5))  # Resultado: 120

Árboles: Estructura de Datos Jerárquica

Un árbol es una estructura de datos jerárquica compuesta por nodos conectados por aristas, donde existe un nodo raíz y cada nodo puede tener cero o más hijos.

Tipos de árboles:

  • Árbol binario: Cada nodo tiene máximo 2 hijos
  • Árbol binario de búsqueda: Árbol binario ordenado
  • Árbol AVL: Árbol binario auto-balanceado
  • Árbol B: Árbol de múltiples hijos usado en bases de datos

Implementación de un Árbol Binario

class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.izquierdo = None
        self.derecho = None

class ArbolBinario:
    def __init__(self):
        self.raiz = None
    
    def insertar(self, valor):
        if self.raiz is None:
            self.raiz = Nodo(valor)
        else:
            self._insertar_recursivo(self.raiz, valor)
    
    def _insertar_recursivo(self, nodo, valor):
        if valor < nodo.valor:
            if nodo.izquierdo is None:
                nodo.izquierdo = Nodo(valor)
            else:
                self._insertar_recursivo(nodo.izquierdo, valor)
        else:
            if nodo.derecho is None:
                nodo.derecho = Nodo(valor)
            else:
                self._insertar_recursivo(nodo.derecho, valor)

Recorridos de Árboles

def recorrido_inorden(nodo):
    if nodo:
        recorrido_inorden(nodo.izquierdo)
        print(nodo.valor)
        recorrido_inorden(nodo.derecho)

def recorrido_preorden(nodo):
    if nodo:
        print(nodo.valor)
        recorrido_preorden(nodo.izquierdo)
        recorrido_preorden(nodo.derecho)

def recorrido_postorden(nodo):
    if nodo:
        recorrido_postorden(nodo.izquierdo)
        recorrido_postorden(nodo.derecho)
        print(nodo.valor)

Aplicaciones Prácticas

  1. Sistemas de archivos: Estructura jerárquica de carpetas
  2. Árboles de decisión: Machine learning y análisis
  3. Parsers: Análisis sintáctico de código
  4. Bases de datos: Índices B-tree para búsquedas eficientes

Complejidad Temporal

  • Búsqueda en árbol binario balanceado: O(log n)
  • Inserción: O(log n)
  • Eliminación: O(log n)
  • Recorrido completo: O(n)

La recursión y los árboles son conceptos fundamentales que aparecen en muchos algoritmos avanzados y son esenciales para entender estructuras de datos más complejas.

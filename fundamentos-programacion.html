Fundamentos de Programación

Variables, funciones y control de flujo

Variables y Tipos de Datos

Las variables son contenedores que almacenan valores que pueden cambiar durante la ejecución del programa.

Tipos de datos básicos:

# Enteros
edad = 25
contador = 0

# Flotantes
precio = 19.99
pi = 3.14159

# Cadenas
nombre = "Juan"
mensaje = 'Hola mundo'

# Booleanos
es_estudiante = True
esta_activo = False

# Listas
numeros = [1, 2, 3, 4, 5]
nombres = ["Ana", "Luis", "María"]

Funciones

Las funciones son bloques de código reutilizable que realizan una tarea específica.

Estructura de una función:

def nombre_funcion(parametros):
    """Documentación de la función"""
    # Cuerpo de la función
    return resultado

# Ejemplo práctico
def calcular_area_rectangulo(base, altura):
    """Calcula el área de un rectángulo"""
    area = base * altura
    return area

# Uso de la función
resultado = calcular_area_rectangulo(5, 3)
print("El área es:", resultado)  # El área es: 15

Control de Flujo

Estructuras condicionales:

# if-elif-else
nota = 85

if nota >= 90:
    calificacion = "A"
elif nota >= 80:
    calificacion = "B"
elif nota >= 70:
    calificacion = "C"
else:
    calificacion = "F"

print("Tu calificación es:", calificacion)

Bucles:

# Bucle for
for i in range(5):
    print("Iteración", i)

# Bucle while
contador = 0
while contador < 5:
    print("Contador:", contador)
    contador += 1

# Bucle for con listas
frutas = ["manzana", "banana", "naranja"]
for fruta in frutas:
    print("Me gusta la", fruta)

Buenas Prácticas

  1. Nombres descriptivos: Usa nombres claros para variables y funciones
  2. Comentarios: Documenta tu código
  3. Funciones pequeñas: Una función debe hacer una sola cosa
  4. Evita repetición: Principio DRY (Don't Repeat Yourself)
  5. Manejo de errores: Usa try-except para errores potenciales

Ejemplo de código bien estructurado:

def es_numero_primo(numero):
    """
    Determina si un número es primo
    Args: numero (int): El número a verificar
    Returns: bool: True si es primo, False en caso contrario
    """
    if numero < 2:
        return False
    
    for i in range(2, int(numero ** 0.5) + 1):
        if numero % i == 0:
            return False
    
    return True

def encontrar_primos(limite):
    """Encuentra todos los números primos hasta un límite"""
    primos = []
    for num in range(2, limite + 1):
        if es_numero_primo(num):
            primos.append(num)
    return primos

# Uso de las funciones
numeros_primos = encontrar_primos(20)
print("Números primos hasta 20:", numeros_primos)
# Resultado: [2, 3, 5, 7, 11, 13, 17, 19]

Estos fundamentos son la base sobre la cual se construyen algoritmos más complejos y estructuras de datos avanzadas. Dominar estos conceptos es esencial para cualquier programador.
